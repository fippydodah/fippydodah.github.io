Projects

Local LLM Voice Assistant

I made a voice assistant that can operate offline. I personally feel unsettled having a voice assistant such as Alex or Google listening to me all day collecting data, so I decided to make my own. This project uses Ollama for the LLM specifically running llama2-uncensored because why not! When the program runs it listens for the set wake word and once it is picked up it lets you know it is listening for a prompt. Using OpenAi's whisper models your prompt is converted into text and then fed into the LLM. The LLM then will start printing out text to let you know it is working. Finally, once the LLM finishes its response the text is spoken to the user.
